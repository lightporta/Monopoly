## 一、游戏概述

### 1.1 游戏背景

本游戏以山东烟台文旅为主题，玩家在36格环形棋盘上掷骰移动、购买地标地产、建造房屋收取过路费，并融合"八仙过海"传送、烟台美食收集、"仙境铁三角"成就胜利等特色玩法。

### 1.2 游戏目标

玩家通过购买、建造、交易等手段获取资产，最终实现以下两种胜利条件之一：
- **破产胜利**：使其他所有玩家破产
- **铁三角胜利**：同时持有烟台山、蓬莱阁、养马岛三处核心地标

### 1.3 玩家人数

支持2-4人游玩，可选择真人对战或人机对战模式。

### 1.4 游戏时间

单局约15-30分钟。

## 二、棋盘结构

### 2.1 棋盘布局

棋盘为10×10网格，共36个格子，呈环形排列（四角共享）：
- 第一行（顶边）：格子0-9
- 第一列（右边）：格子9-19
- 最后一行（底边）：格子19-28（反向）
- 最后一列（左边）：格子28-35（反向）

### 2.2 格子类型

| 类型 | 数量 | 说明 |
|------|------|------|
| 起点（start） | 1 | 仙境海岸，经过获得2000元奖励并抽美食卡 |
| 地产（property） | 21 | 可购买的地标地产 |
| 机会（chance） | 3 | 抽取机会卡 |
| 命运（destiny） | 3 | 抽取命运卡 |
| 传送（teleport） | 2 | 烟台山灯塔→蓬莱阁，八仙渡海口→任意空地 |
| 移动（move） | 2 | 海昌鲸鲨馆→再掷一次，机场→任意格 |
| 休息（rest） | 1 | 养马岛跨海大桥，跳过下一回合 |

## 三、游戏流程

### 3.1 初始设置

1. 每位玩家获得初始资金15000元
2. 所有玩家从起点（格子0）出发
3. 机会卡和命运卡堆洗牌备用
4. 由系统随机决定先手玩家

### 3.2 回合流程

```
开始回合 → 操作资产（建造/抵押/赎回）→ 掷骰子 → 移动棋子 → 触发格子事件 → 结束回合
```

### 3.3 特殊情况

#### 3.3.1 三连对子判定
- 连续掷出三次双数（对子），直接传送至休息格（格子18）
- 跳过下一回合，不领取经过起点奖励
- 连击计数重置

#### 3.3.2 过起点奖励
- 棋子经过或停在起点时，获得2000元奖励
- 同时抽取一张美食卡

#### 3.3.3 回合跳过
- 停在休息格或触发三连对子时，下回合跳过
- 跳过回合不消耗任何操作

## 四、格子事件细则

### 4.1 起点格（格子0）

- 停留：无特殊奖励
- 经过：获得2000元 + 抽美食卡

### 4.2 地产格

#### 4.2.1 无主地产
- 玩家可选择购买或放弃
- 购买价格为地产面值
- 放弃后触发拍卖流程

#### 4.2.2 有主地产
- 若是自己的地产：无事件
- 若是他人地产：支付过路费
- 可使用免租券免付过路费

#### 4.2.3 拍卖流程
- 起拍价为地价的50%
- 每次加价≥100元
- 倒计时10秒无人加价则成交
- 无人参与则流拍（地产保持无主）

### 4.3 机会格

抽取机会卡，触发卡片效果（通常为正面效果）：
- 现金奖励
- 移动指定步数
- 传送至特定位置
- 抽美食卡

### 4.4 命运格

抽取命运卡，触发卡片效果（可能为负面效果）：
- 现金增减
- 移动指定步数
- 送入休息格

### 4.5 传送格

#### 4.5.1 烟台山灯塔（格子5）
- 固定传送至蓬莱阁（格子24）
- 传送后触发蓬莱阁的格子事件
- 不经过起点

#### 4.5.2 八仙渡海口（格子25）
- 选择传送至任意空地
- 传送后若空地无主，可自动购买

### 4.6 移动格

#### 4.6.1 海昌鲸鲨馆（格子13）
- 获得一次额外掷骰子机会

#### 4.6.2 烟台蓬莱国际机场（格子35）
- 选择传送至任意格子（含已售）

### 4.7 休息格（格子18）

- 养马岛跨海大桥
- 跳过下一回合

## 五、经济与交易

### 5.1 地产购买

- 玩家可在自己回合掷骰前或掷骰后停在无主地产时购买
- 购买价格为地产面值
- 购买后成为地产所有者

### 5.2 建造房屋

- 建造费用 = 地价 × 0.4
- 每块地产最多可建造3栋房屋 + 1座旅馆
- 同色块集齐后才能建造

### 5.3 抵押

- 抵押可得 = 地价 × 0.5
- 抵押中的地产不收过路费
- 同色块加成不因抵押失效

### 5.4 赎回

- 赎回费用 = 抵押金 × 1.1
- 赎回后恢复收租权

### 5.5 破产

#### 5.5.1 破产条件
- 现金不足以支付债务（过路费等）

#### 5.5.2 破产清算流程
1. 自动变卖所有建筑（返还建筑成本的50%）
2. 自动抵押所有地产（获得抵押金）
3. 若仍不足以支付，宣告破产

#### 5.5.3 破产后果
- 破产玩家退出游戏
- 所有地产归还银行变为无主
- 剩余玩家继续游戏

## 七、UI交互要求

### 7.1 棋盘视觉
- 36格环形布局，使用CSS Grid实现
- 地产格显示色块颜色和建筑图标
- 玩家棋子显示在当前位置

### 7.2 动画效果
- 骰子动画：800ms
- 棋子移动：300ms/格
- 卡牌翻转：400ms
- 弹窗淡入：200ms

### 7.3 弹窗优先级
- 模态层级：事件弹窗 < 选择弹窗 < 系统弹窗（破产/胜利）
- 禁止点击背景关闭弹窗

### 7.4 响应式设计
- ≥1024px：桌面版完整布局
- 768~1023px：棋盘缩小、面板折叠为抽屉
- <768px：纵向布局，棋盘居中、面板上下排列

## 八、胜负结算

### 8.1 破产胜利
- 当只剩一名玩家未破产时，该玩家获胜
- 获胜者获得所有剩余资产

### 8.2 铁三角胜利
- 同时持有烟台山、蓬莱阁、养马岛三处地产
- 抵押状态不影响胜利判定（只要所有权仍属于该玩家）
- 达成铁三角后立即获胜

### 8.3 结算页面
- 显示获胜者信息
- 显示所有玩家的资产排名
- 提供"再来一局"按钮

## 十、边界规则（补充说明）

### 10.1 操作时机
- 玩家在自己回合掷骰前可执行任意数量的建造、抵押、赎回操作
- 掷骰后本回合只能触发格子事件，不能再操作资产

### 10.2 三连对子
- 第三次掷出对子时不触发第三次移动事件，直接传送至格18
- 不领取经过起点奖励

### 10.3 抵押状态
- 抵押中地产不收过路费，但同色块集齐加成仍按地产归属判定

### 10.4 铁三角胜利
- 需同时持有三处地产的所有权
- 抵押状态不影响胜利判定

### 10.5 传送格事件触发
- 烟台山灯塔传送到蓬莱阁后，触发蓬莱阁的格子事件
- 若蓬莱阁已被对手拥有且已集齐仙山5块，触发色块加成

### 10.6 美食卡兑换时机
- 仅在自己回合掷骰前可兑换

### 10.7 破产后色块加成
- 玩家破产后地产归银行变为无主
- 对手此前依赖该玩家持有某色块而激活的加成失效
